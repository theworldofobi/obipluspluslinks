## Obi++ Resource Guide for C++ Developers

Welcome to my official repository for resources on becoming a proficient junior-level C++ engineer. Below are a list of book, video, and website recommendations of useful information to get started. They are ordered roughly by level of complexity within their respective categories. Prefer to read through the written content thoroughly and use the videos as content primers.Â 

Be sure to get creative and think of ways to apply what you have learned into meaningful projects. Happy learning!
